# 🌐 Antigravity 반도체 주니어 포트폴리오 하네스 템플릿

이 템플릿은 **반도체 후공정(Packaging & Test) 분야의 0년차 신입(Junior) 엔지니어**가 자신의 학술 성과, 공정 장비 실습 경험, 분석 기술 역량을 시각적으로 극대화하여 기업 면접관의 눈길을 사로잡는 프리미엄 포트폴리오 웹사이트를 개발할 수 있도록 돕는 프레임워크 가이드라인입니다.

---

## 📂 템플릿 구성 요소

본 디렉토리(`design_harness_template`)는 반도체 도메인에 특화된 5개의 마스터 문서로 구성되어 있습니다. 이를 프로젝트 루트 경로에 배치하여 활용하세요.

1. **`requirements.md` (요구사항 정의서)**
   - 반도체 후공정 주니어 포트폴리오가 담아야 할 **4대 핵심 기능**(인터랙티브 공정 흐름도, 장비 대시보드, 프로젝트 카드 그리드, 연락처 모달)과 페이지 정보 구조를 설계합니다.
2. **`design_system.md` (디자인 시스템 가이드라인)**
   - 실리콘 그레이배경, 초정밀 클린룸 시안 블루, 열처리 오렌지 빛으로 어우러지는 **'Silicon & Cyan'** HSL 색상 공간과 미세 회로 패턴 배경 CSS, LED indicator 인터랙션 스펙을 포함합니다.
3. **`technical_stack.md` (기술 스택 및 구현 규칙)**
   - HTML5/CSS/ES6+ Vanilla JS 규칙, 네이티브 `<dialog>` 팝업 구현 방식, SEO 메타태그 구성 및 `generate_image`를 활용하여 독창적인 반도체 3D 그래픽을 만드는 템플릿 프롬프트를 담고 있습니다.
4. **`prompt_harness.md` (프롬프트 하네스 마스터)**
   - 포트폴리오 개발 시작 시 Antigravity에게 첫 번째로 복사해 전달하는 마스터 프롬프트입니다. AI가 스스로 반도체 지식을 보유한 수석 엔지니어 멘토 역할을 수행하며 코드 완성도를 극대화하도록 강제합니다.
5. **`README.md` (본 가이드)**
   - 템플릿의 전체 아키텍처 및 반도체 신입 포트폴리오 핵심 설계 팁을 담은 종합 안내서입니다.

## 🔬 주니어 반도체 엔지니어 포트폴리오 설계 핵심 팁

1. **경력 공백을 극복하는 '장비/분석 능력' 시각화**
   - 신입 지원자는 실무 경력이 없기 때문에, 대학 전공 실험이나 외부 전문 교육(예: 반도체 8대 공정 실습)에서 직접 **레시피를 잡고 돌려본 장비 경험**을 자세히 부각해야 합니다.
   - 예: "Sputter 장비 공정 압력 설정 경험", "SEM(주사전자현미경) 분석을 통한 패키지 보이드(Void) 크기 측정".
2. **후공정 도메인 전문성 입증**
   - 최근 반도체 미세화 한계로 인해 Packaging 기술(Flip-chip, TSV, Fan-Out, SiP 등)의 중요성이 급격히 부각되고 있습니다. 
   - [requirements.md](file:///Users/oasunryo/Documents/antigravity/design_harness_template/requirements.md)의 공정 흐름도 섹션을 활용하여 본인이 관심 있는 후공정 핵심 기술(예: 와이어 본딩의 Cu 와이어 적용 시 신뢰성 이슈 분석 등)을 상세히 담으세요.
