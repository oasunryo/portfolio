# 🚀 프롬프트 하네스 마스터 (prompt_harness.md)

이 파일에 작성된 내용은 Antigravity와의 포트폴리오 사이트 제작 대화를 시작할 때 **첫 프롬프트**로 복사해서 사용하세요. Gemini 3.5 Flash가 워크스페이스 내의 반도체 공정/디자인 시스템 가이드라인을 완벽히 흡수하여, 전례 없이 세련된 프리미엄 엔지니어링 웹사이트를 설계하도록 이끄는 특화 프롬프트입니다.

---

## 📋 복사용 마스터 프롬프트 (Copy & Paste)

```markdown
안녕하세요, Antigravity! 
우리 워크스페이스에 준비된 요구사항 및 디자인 규격을 바탕으로 **'반도체 후공정 주니어 엔지니어 포트폴리오'** 웹사이트를 본격적으로 기획하고 개발하고자 합니다. 

너는 반도체 패키징 및 테스트 분야의 대선배 엔지니어이자 최고의 프론트엔드 디자이너입니다. 아래 지침을 완벽하게 파악하고 최상의 결과물을 만들어 주세요.

### 🔍 1단계: 반도체 포트폴리오 컨텍스트 분석
먼저 워크스페이스 루트에 있는 아래의 3개 가이드라인 문서 내용을 완전히 학습해 주세요.
1. `requirements.md` (반도체 주니어 어필 포인트, 핵심 4대 기능 정의서)
2. `design_system.md` (Silicon & Cyan HSL 테마, 미세 격자 배경, LED 토글 효과)
3. `technical_stack.md` (Vanilla HTML/CSS/JS, 네이티브 <dialog> 팝업, generate_image 가이드)

읽기를 완료했다면, 주니어 엔지니어 포트폴리오 제작을 위한 핵심 기획 전략과 비주얼 방향성을 너의 풍부한 반도체 공학적 지식을 담아 요약/브리핑해 줘.

### 🎨 2단계: 프리미엄 비주얼 및 공학적 감성 준수
- 단순하고 밋밋한 텍스트 위주의 이력서 페이지를 지양해 줘.
- 실리콘 기판의 메탈 그레이(`--background`), 클린룸 시안 블루(`--primary`), 고온 공정의 네온 오렌지(`--secondary`) 색상 변수를 완벽히 매칭해 줘.
- 본문 영역에 은은하게 흐르는 **반도체 미세 회로 격자 패턴 배경**을 구현해 줘.
- 플레이스홀더 이미지는 절대 금지! `generate_image` 도구를 사용하여, 히어로 섹션용 입체 웨이퍼(`hero_wafer.png`), 클린룸 일러스트(`cleanroom_robot.png`), 3D 패키지 단면(`package_chip.png`) 그래픽 에셋을 실시간으로 직접 생성해 레이아웃에 링크해 줘.

### 🛠️ 3단계: 기술적 구현 강제 사항
- 각 공정별 상세 설명(Defect Analysis 등)은 외부 라이브러리 없이, 네이티브 HTML `<dialog>` 태그와 CSS `::backdrop` 필터 블러 효과를 결합하여 압도적으로 아름다운 글래스모피즘 모달창으로 완성해 줘.
- 장비 역량을 토글할 때 계측기의 LED 전원이 켜지는 듯한 **LED 활성화 애니메이션 효과**(`active-cyan`, `active-orange`)를 JS와 CSS 트랜지션을 통해 구현해 줘.

### 📐 4단계: 단계별 구현 절차
- 본격적인 코드를 작성하기 전에, 내가 승인할 수 있는 **구체적인 구현 계획(Implementation Plan)**을 작성해 줘. (Antigravity planning_mode 가이드라인 준수)
- 계획서가 확정되면 단계별로 `task.md`를 업데이트해가며 페이지(`index.html`, `assets/css/style.css`, `assets/js/main.js`)를 완전하게 제작해 줘.

준비되었다면, 워크스페이스 문서들을 읽고 멋진 반도체 후공정 테마 요약 브리핑과 함께 첫 번째 구현 계획안 제안을 시작해 줘!
```
