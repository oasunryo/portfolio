# 🚀 Notion API 연동 Next.js 프리미엄 포트폴리오 웹사이트

이 프로젝트는 **Notion 데이터베이스**를 백엔드 CMS로 연동하여 나만의 프로젝트 경험을 효율적으로 관리하고, **Next.js**를 활용하여 웹사이트를 평생 무료로 운영할 수 있도록 설계된 프리미엄 다크 테마 포트폴리오 템플릿입니다.

노션 데이터베이스에서 새로운 프로젝트를 추가하거나 설정을 수정하면, Next.js의 **ISR(Incremental Static Regeneration - 점진적 정적 재생성)** 기술을 통해 수초~수분(기본 설정 60초) 이내에 웹페이지에 자동으로 반영됩니다.

---

## ✨ 주요 기능 및 특징
*   **실시간 자동 업데이트 (ISR):** 별도의 재빌드나 배포 과정 없이 노션에서 데이터를 관리하면 사이트가 자동으로 갱신됩니다.
*   **프리미엄 바닐라 CSS 디자인:** 글래스모피즘(Acrylic Glassmorphism) 질감 효과, 네온 퍼플/시안 그라디언트 테두리, 부드러운 호버 애니메이션이 적용되어 시각적으로 매우 수려합니다.
*   **하이브리드 카테고리 필터링 & 검색:** 사용자가 원하는 키워드(태그, 제목, 설명 등)로 프로젝트를 즉시 검색하고 카테고리별로 필터링할 수 있습니다.
*   **인터랙티브 모달 상세 뷰:** 카드를 클릭하면 상세 기여도 정보 및 마크다운 본문을 모달창으로 깔끔하게 보여줍니다.
*   **유연한 샌드박스 폴백(Fallback):** 노션 연동 토큰이 없거나 만료된 상태에서도 즉시 작동하는 데모 데이터(Mock Data)가 기본 내장되어 있어 에러 없이 로컬 실행 및 확인이 가능합니다.

## 🛠️ Notion 데이터베이스 연동 가이드

나만의 노션 데이터베이스를 연동하고 싶으시다면 아래 절차를 천천히 따라해 주세요.

### Step 1. Notion API 프라이빗 통합 키 발급
1. [Notion API 개발자 포털(developers.notion.com)](https://developers.notion.com/)에 로그인합니다.
2. **[내 통합 (My Integrations)]** -> **[새 통합 만들기 (Create new integration)]**를 클릭합니다.
3. 유형은 **프라이빗(Private)**으로 선택하고 적절한 이름을 입력해 저장합니다.
4. 완성 후 화면에 표시되는 **프라이빗 API 통합 토큰 (Secret API Key)**을 안전하게 복사해 둡니다.

### Step 2. 포트폴리오 데이터베이스 페이지 생성 및 설정
노션에 새 페이지를 생성한 뒤 **[데이터베이스 - 인라인]** 또는 **[데이터베이스 - 전체 페이지]**를 생성하고 다음과 같은 **정확한 명칭과 타입**의 속성(Properties)을 만들어 줍니다.

| 속성 이름 (Name) | 속성 유형 (Type) | 상세 설명 (Description) |
| :--- | :--- | :--- |
| **Name** | 제목 (Title) | 프로젝트/경험 이름 (예: `스마트 캘린더 플랫폼 개발`) |
| **Category** | 선택 (Select) | 대분류 태그 (예: `개발`, `디자인`, `기획`, `마케팅` 등) |
| **Period** | 텍스트 (Text) | 진행 기간 (예: `2026.01 - 2026.04`) |
| **Role** | 텍스트 (Text) | 역할 및 본인 기여도 (예: `Front-end Developer (1인 개발)`) |
| **Tags** | 다중 선택 (Multi-select) | 기술 스택 및 키워드 (예: `React`, `Next.js`, `Figma`) |
| **Description** | 텍스트 (Text) | 카드 그리드에 표시될 짧은 줄글 요약 설명 |
| **Content** | 텍스트 (Text) | 모달 창에 나타날 Markdown 형식의 상세 프로젝트 내용 |
| **Link** | URL | 관련 링크 또는 GitHub 링크 (선택 사항) |
| **Badge** | 텍스트 (Text) | 강조 배지 텍스트 (예: `Featured`, `Best Strategy`) (선택 사항) |

### Step 3. 데이터베이스와 API 통합 연결
1. 노션에서 방금 만든 데이터베이스 페이지로 이동합니다.
2. 페이지 우측 상단의 `...` (더 보기) 아이콘을 클릭합니다.
3. 가장 아래에 있는 **[연결 추가 (Add connections)]**를 클릭하고, Step 1에서 만든 통합 서비스의 이름을 검색하여 연결을 허용합니다.

### Step 4. 데이터베이스 ID 추출
*   데이터베이스 페이지의 주소(URL)를 확인합니다.
*   예시: `https://www.notion.so/myworkspace/8b4886b6279f42778da5ad4694936d8b?v=...`
*   여기서 `myworkspace/`와 물음표(`?`) 사이의 **32자리 알파벳/숫자 조합**(`8b4886b6279f42778da5ad4694936d8b`)이 바로 **데이터베이스 ID**입니다.

### Step 5. 환경 변수 파일(`.env.local`) 생성 및 입력
프로젝트 루트 디렉토리에 `.env.local` 파일을 생성하고 발급받은 두 가지 값을 입력합니다.

```env
NOTION_TOKEN=secret_your_notion_integration_token_here
NOTION_DATABASE_ID=your_32_digit_database_id_here
```

## 🏃 로컬 실행 및 검증 방법

### 1. 개발 서버 구동
작업 공간 터미널에서 다음 명령어를 실행하여 로컬 서버를 가동합니다:
```bash
npm run dev
```
가동 후 웹 브라우저에서 `http://localhost:3000`으로 접속하면 세련된 글래스모피즘 테마의 동적 포트폴리오를 바로 만나보실 수 있습니다.

### 2. 빌드 테스트
프로덕션용 최종 빌드 검증은 아래 명령어로 수행합니다:
```bash
npm run build
```

## 🛡️ Vercel로 평생 무료 배포하기
1. 완성된 코드를 본인의 **GitHub 저장소(Repository)**에 푸시합니다.
2. [Vercel 공식 웹사이트(vercel.com)](https://vercel.com/)에 가입한 뒤 GitHub 계정을 연동합니다.
3. **Add New Project**를 선택하고 해당 저장소를 가져옵니다.
4. 배포 설정 중 **Environment Variables** 영역에 다음 두 가지를 동일하게 입력합니다:
   * `NOTION_TOKEN`
   * `NOTION_DATABASE_ID`
5. **[Deploy]** 버튼을 누르면 단 30초 만에 나만의 포트폴리오 웹사이트가 인터넷에 정상 무료 배포됩니다!
